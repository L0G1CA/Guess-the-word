# Guess-the-word

This is my first project i had to make for school.

You have 7 lives. everytime you guess wrong, you lose a live.
you may guess a letter or the word at a attempt.

there is a challenge mode.
there are 4 challenges that you can all use at the same time.

currently, all words are in dutch as this is a school project for my dutch school.

Play the game at: [text](https://l0g1ca.github.io/Guess-the-word/)
